# 📊 CLIENT HEALTH SCORECARD — AusAI Tech

**Purpose:** Score every active client on 5 dimensions to flag at-risk relationships before they churn.  
**When:** Monthly (first week of each month).  
**Time:** 15 min per active client.

> **Why this matters:** It costs 5× more to acquire a new client than retain an existing one. A client scoring 3/5 on payment timeliness hasn't churned *yet* — but they will. Catch it early.

## 1. THE 5 SCORING DIMENSIONS

| # | Dimension | What It Measures |
|---|---|---|
| 1 | **Payment Timeliness** | Do they pay on time? How many reminders needed? |
| 2 | **Scope Stability** | How many change orders? Scope creep frequency? |
| 3 | **Communication Quality** | Clear, respectful, responsive? Or vague, demanding, silent? |
| 4 | **Referral Potential** | Have they referred? Do they talk about you positively? |
| 5 | **Growth Potential** | Can they be upsold? Projects expanding? Budget growing? |

Each dimension scored **1 (bad) to 5 (excellent)**. Total score: 5–25.

---

## 2. SCORING RUBRIC

### Dimension 1: Payment Timeliness

| Score | Criteria |
|---|---|
| **5** | Pays within 24 hrs of invoice. Never needs a reminder. |
| **4** | Pays within 3 days. Occasional reminder. |
| **3** | Pays within 7 days. Needs 1 reminder most invoices. |
| **2** | Pays late (>7 days). Needs 2+ reminders. Has disputed a charge. |
| **1** | Currently overdue. Has threatened non-payment. Payment plan requested. |

### Dimension 2: Scope Stability

| Score | Criteria |
|---|---|
| **5** | Zero scope changes outside Change Orders. Respects boundaries perfectly. |
| **4** | 1 minor scope request (handled via Change Order). No friction. |
| **3** | 2-3 scope requests. Some resistance to paying for extras. |
| **2** | Frequent "can you just…" requests. Pushes back on Change Orders. |
| **1** | Scope creep is constant. Refuses Change Orders. "But you said you'd automate everything." |

### Dimension 3: Communication Quality

| Score | Criteria |
|---|---|
| **5** | Clear briefs, fast replies (<4 hrs), appreciative, respectful tone. |
| **4** | Generally clear. Replies within 24 hrs. Professional tone. |
| **3** | Vague instructions sometimes. Replies take 1-2 days. Neutral tone. |
| **2** | Unclear briefs requiring clarification loops. Goes silent for 3+ days. Slightly demanding. |
| **1** | Hostile, belittling, or ghosting. "Just figure it out." Late-night/weekend demands. Blames you for their delays. |

### Dimension 4: Referral Potential

| Score | Criteria |
|---|---|
| **5** | Has actively referred 1+ paying clients. Gave unsolicited testimonial. Tags you on LinkedIn. |
| **4** | Has verbally said they'd refer. Gave testimonial when asked. |
| **3** | Happy with work but hasn't referred. Would probably say yes if asked. |
| **2** | Mixed feelings. Might not recommend you. Hasn't responded to testimonial requests. |
| **1** | Unhappy. Has complained to others. Negative review risk. |

### Dimension 5: Growth Potential

| Score | Criteria |
|---|---|
| **5** | Already on retainer. Discussing Phase 2. Budget expanding. |
| **4** | Expressed interest in retainer/upsell. Single project but open to more. |
| **3** | One-off project. No discussion of future work yet. |
| **2** | Explicitly said "just this one project." Budget-constrained. |
| **1** | Wants to cancel/downgrade. "We're bringing this in-house." |

## 4. INTERPRETING SCORES

| Total Score | Tier | Label | Action |
|---|---|---|---|
| **21–25** | 🟢 Green | Champion | Protect + nurture. These are your best clients. |
| **16–20** | 🟡 Yellow | Stable | Good. Check monthly. Look for upsell opportunities. |
| **11–15** | 🟠 Orange | At-Risk | Needs attention THIS MONTH. Address the lowest dimension. |
| **5–10** | 🔴 Red | Critical | Immediate intervention or prepare to offboard. |

### Red Flag Rules

**Any single dimension scoring 1** = automatic review regardless of total score.

**Score drops 4+ points from last month** = emergency check-in call.

## 5. ACTION PLAYBOOKS BY SCORE

### 🟢 Champion (21-25)

- **Send a surprise thank-you:** Handwritten note, small gift (see `CLIENT_GIFT_GUIDE.md`), or free audit add-on.
- **Ask for a referral NOW:** "You've been great to work with — do you know anyone else who might benefit from automation?"
- **Invite to case study:** This is your best Case Study candidate.
- **Introduce new service ideas:** They trust you. Pitch the retainer.
- **Frequency:** Monthly check-in. Quarterly in-person coffee/lunch.

### 🟡 Stable (16-20)

- **Identify the lowest dimension:** What's keeping them from Champion?
- **Proactive value-add:** Share a relevant article, tool tip, or free workflow suggestion.
- **Upsell gently:** "We built X. Have you thought about automating Y too?"
- **Frequency:** Monthly email. Quarterly video call.

### 🟠 At-Risk (11-15)

- **Investigate the lowest dimension immediately:**
  - Payment issues → Send gentle reminder. Check if they're satisfied with value delivered.
  - Scope creep → Revisit the SOW. Offer a Change Order to formalise extras.
  - Communication → Schedule a "project health check" call. Ask: "How's the project working for you?"
  - Low referral potential → Ask for feedback: "Is there anything we could do better?"
- **Document everything:** If this goes Red, you need a paper trail.
- **Frequency:** Bi-weekly check-in. Weekly during active project.

### 🔴 Critical (5-10)

- **Immediate intervention call:** "I want to make sure we're aligned and you're happy with the work."
- **Identify the root cause:** Is it fixable? (Miscommunication, scope mismatch, payment process friction)
- **Offer a remedy:**
  - Scope issue → Re-scope. Offer discount on next phase if the fault is partly yours.
  - Payment issue → Offer payment plan. Check if they're facing cash flow issues.
  - Communication breakdown → Reset expectations. Weekly sync calls.
- **Prepare offboarding:** If they're scoring 1 on multiple dimensions and unresponsive to interventions, start the offboarding process (`CLIENT_OFFBOARDING.md`).
- **DO NOT upsell a Red client.** Fix the relationship first.
- **Frequency:** Weekly until score improves. If no improvement in 30 days, exit gracefully.

## 6. MONTHLY REVIEW PROTOCOL

### First Monday of Each Month (30 min)

1. **Score every active client** using the rubric above
2. **Update the scorecard** (save in client folder or spreadsheet)
3. **Compare to last month:** Identify trends (improving, stable, declining)
4. **Prioritise actions:**
   - Red clients: Book intervention calls THIS WEEK
   - Orange: Send check-in messages + identify fixes
   - Yellow: Look for upsell triggers
   - Green: Send thank-you + referral ask
5. **Track metrics:**
   - Average client health score (target: >18)
   - Number of Red clients (target: 0)
   - Churn rate (target: <10%/year)
   - Referral rate (target: 1+ per Green client/year)

### Annual Review

- Promote/demote clients between tiers
- Drop clients who averaged <12 for 3+ consecutive months
- Increase prices for Champions (they'll pay)
