## 1. THE 5 SHORTS FORMATS

### Format 1: "Before → After" (Highest Converting)

```
0-3 sec:   "This client spent 15 hours/week doing THIS." (show manual process)
3-10 sec:  Screen recording of the actual automation running
10-45 sec: "Here's how I built it:" (3-step walkthrough with text overlays)
45-55 sec: Result: "{X} hours saved per week. {Y}% fewer errors."
55-60 sec: "Want this for your business? Link in bio."
```

**Why it works:** Concrete proof. Specific numbers. Visual transformation.

### Format 2: "Tool Tip" (Highest Search Traffic)

```
0-3 sec:   "One Zapier trick that saves 5+ hours/week."
3-10 sec:  Show the problem setup (before)
10-45 sec: Step-by-step setup (screen recording)
45-55 sec: "This takes 10 minutes to set up. Saves 5 hours every week."
55-60 sec: "Follow for more automation tips."
```

**Why it works:** Searchable. Evergreen. Shares well in communities.

### Format 3: "Common Mistake" (Highest Engagement)

```
0-3 sec:   "Stop building automations without THIS." (controversial hook)
3-10 sec:  Explain the common mistake (name it explicitly)
10-45 sec: Show what happens when you make the mistake + the fix
45-55 sec: "I've fixed this for 40+ clients. Don't learn the hard way."
55-60 sec: "Save this for your next automation build."
```

**Why it works:** Triggers "oh no, am I doing this?" — high save rate.

### Format 4: "Stack Compare" (Authority Builder)

```
0-3 sec:   "Zapier vs Make vs n8n — which one actually?"
3-15 sec:  Quick visual comparison of each (1 screen each)
15-45 sec: "Choose Zapier if {criteria}. Choose Make if {criteria}. Choose n8n if {criteria}."
45-55 sec: "I use all three — depends on the job. Which one's your stack?"
55-60 sec: "Full comparison in my profile link."
```

**Why it works:** High search volume. Positions you as the expert.

### Format 5: "Result Flex" (Social Proof Builder)

```
0-3 sec:   "Client message from this morning:" (show screenshot)
3-10 sec:  Read the testimonial/result out loud
10-30 sec: Show what you built that produced the result
30-55 sec: "This is a {industry} business. They were doing {X} manually. Now it's automated."
55-60 sec: "If you want results like this → link in bio."
```

**Why it works:** Client proof trumps claims. Human emotion + specific result.

## 2. SCRIPT FRAMEWORKS

### The 60-Second Structure

```
0-3 sec:   HOOK — Make them stop scrolling
3-10 sec:  PROBLEM — Why this matters
10-45 sec: SOLUTION — How you fix it (with visuals)
45-55 sec: RESULT — What happens after
55-60 sec: CTA — What to do next
```

### Hook Formulas (first 3 seconds = everything)

| Type | Example |
|---|---|
| **Number** | "3 automations every business needs by 2026" |
| **Controversy** | "Stop paying $30/month for Zapier" |
| **Curiosity** | "This automation runs while I sleep" |
| **Pain** | "If you're still doing THIS manually, stop" |
| **Result** | "I saved a client 22 hours last week. Here's how" |
| **Question** | "How many hours are you losing to repetitive tasks?" |

### Voice-Over Script Template

```
[HOOK - on camera or bold text]
"{hook_line}"

[SCREEN RECORDING - automation in action]
"{explain_what_theyre_seeing}"
"Notice how {key_detail} automatically {action}."

[ZOOM IN - highlight specific step]
"The magic happens here: {technical_but_simple_explanation}"

[RESULT SLIDE - text overlay with numbers]
"Result: {specific_number} {metric}"

[CTA - on camera or end card]
"Want this for your business? {cta}"
```

## 3. RECORDING & EDITING

### Tools

| Tool | Use | Cost |
|---|---|---|
| **OBS Studio** | Screen recording | Free |
| **CapCut** | Editing, text overlays, captions | Free |
| **Canva** | Thumbnails, end cards | Free |
| **Descript** | AI captions + editing | Free tier |
| **Your phone** | On-camera segments (1080p 60fps works) | Free |

### Recording Checklist

- [ ] **Vertical aspect ratio:** 9:16 (1080×1920)
- [ ] **Resolution:** 1080p minimum
- [ ] **Frame rate:** 30fps (60fps optional)
- [ ] **Audio:** Clear voice, no background noise (use phone mic or basic USB mic)
- [ ] **Lighting:** Face well-lit (window light or ring light)
- [ ] **Captions:** Always add auto-captions (CapCut/Descript). 70%+ watch without sound.
- [ ] **Text overlays:** Key numbers, tool names, steps on screen

### Batch Recording Session (2 hrs = 5 Shorts)

```
Min 0-15:  Set up: Open all workflows/tools to show. Scripts ready.
Min 15-25: Record Short 1 (Format: Before→After)
Min 25-35: Record Short 2 (Format: Tool Tip)
Min 35-45: Record Short 3 (Format: Common Mistake)
Min 45-55: Record Short 4 (Format: Stack Compare)
Min 55-65: Record Short 5 (Format: Result Flex)
Min 65-120: Edit all 5 in CapCut (add captions, text, trim silences)
```

## 7. GROWTH TACTICS

### YouTube-Specific

- **First hour engagement:** Reply to every comment within 1 hour of publishing
- **Pin a comment:** "Want this automation for YOUR business? → {calendly_link}"
- **End screen:** Link to your most popular Short or full video
- **Playlist:** Create "AI Automation Shorts" playlist — boosts watch time
- **Shorts shelf:** Optimise thumbnail for the Shorts shelf (bright, clear text, face if possible)

### Cross-Platform Distribution

| Platform | Format | Effort |
|---|---|---|
| YouTube Shorts | Native upload | — (primary) |
| TikTok | Repost same video | 2 min |
| Instagram Reels | Repost same video | 2 min |
| LinkedIn | Repost + text summary | 5 min |
| Twitter/X | Clip as GIF + thread | 10 min |

> **One Short = 5 platforms.** Record once, distribute everywhere.

### Collaboration

- **Comment on other automation creators' Shorts** (genuine, non-spammy)
- **Stitch/Duet** popular automation Shorts with your take
- **Tag tools:** @zapier @makeautomation in posts — they sometimes reshare

## 8. METRICS

### Track Weekly

| Metric | Month 1 | Month 3 |
|---|---|---|
| Shorts published | 3/week | 3/week |
| Avg views per Short | 200-500 | 1,000-5,000 |
| Subscribers gained/week | 5-10 | 20-50 |
| Comments per Short | 2-5 | 5-15 |
| Watch time (hours) | 5-10 | 20-50 |
| CTR (impressions → view) | 3-5% | 5-8% |
| Retention (watched 50%+) | 40-50% | 50-65% |
| Inbound leads from YouTube | 0-1/mo | 2-5/mo |

### What "Good" Looks Like

- **Retention >60%:** Strong signal — YouTube pushes it more
- **CTR >5%:** Thumbnail + title are working
- **Views >1,000 within 48 hours:** Algorithm picked it up — double down on this format
- **Views <100 after 7 days:** Kill the format or improve hook
