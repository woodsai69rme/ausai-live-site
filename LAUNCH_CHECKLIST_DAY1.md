# 📅 30-DAY LAUNCH CHECKLIST — AusAI Tech

**Purpose:** Day-by-day launch plan for your first 30 days as an AI automation consultant.  
**Print this.** Check off each day. Don't skip ahead.

> **Rule:** Days 1-7 are SETUP. Don't chase clients until Day 8. A solid foundation beats frantic outreach.

---

## 📋 WEEK 1: SETUP (Days 1-7)

### Day 1 — Legal Foundation
- [ ] Apply for ABN at abr.gov.au (need TFN — ~15 min)
- [ ] Open business bank account (ING/Up — ~10 min online)
- [ ] Set up tax set-aside: Create a separate savings account. Transfer 25% of every payment here.
- [ ] Read `COMPLIANCE_CHECKLIST.md` Phase 1-3

### Day 2 — Deploy the Site
- [ ] Push pages.yml to GitHub (see `DEPLOYMENT_QUICK_START.md` Step 0)
- [ ] Enable GitHub Pages (Step 1)
- [ ] Verify all 4 pages load at your GitHub Pages URL
- [ ] Buy domain ($10-15) and point to GitHub Pages

### Day 3 — Payment + Booking
- [ ] Create Stripe account (stripe.com)
- [ ] Create 4 products + Payment Links (see `STRIPE_SETUP_GUIDE.md`)
- [ ] Replace placeholder URLs in checkout.html
- [ ] Create Calendly account + 3 event types
- [ ] Replace YOUR_USERNAME in book.html
- [ ] Test: Can someone book a call and pay you?

### Day 4 — Social Profiles
- [ ] Create/optimise LinkedIn profile (headline, about, featured)
- [ ] Create Twitter/X account
- [ ] Create Reddit account (if you don't have one with 50+ karma)
- [ ] Claim your Fiverr account
- [ ] Set up email signature (`EMAIL_SIGNATURE.html`)

### Day 5 — Content Creation
- [ ] Write 7 LinkedIn posts (batch — use templates in `SOCIAL_PROOF_ENGINE.md`)
- [ ] Write 5 Reddit comments to start building karma
- [ ] Record 1 YouTube Short (Format 1: Before→After — see `YOUTUBE_SHORTS_STRATEGY.md`)
- [ ] Set up Fiverr gigs (copy from `FIVERR_GIG_CONTENT.md`)

### Day 6 — Outreach Setup
- [ ] Create LinkedIn prospect list (20 targets — see `LINKEDIN_OUTREACH_SCRIPTS.md`)
- [ ] Set up tracking spreadsheet (`LEAD_TRACKER.html`)
- [ ] Draft first 5 LinkedIn DMs (personalise — don't copy-paste)
- [ ] Set up `COLD_OUTREACH_AB_TEST.md` tracking sheet

### Day 7 — Practice + Review
- [ ] Practice `Discovery_Call_Script.md` out loud (15 min)
- [ ] Review `COMPETITOR_BATTLE_CARDS.md`
- [ ] Print `MASTER_INDEX_1PAGE.md` — stick on wall
- [ ] Open `AUSAI_OPS_DASHBOARD.html` — this is now your daily command centre
- [ ] Review Week 1 checklist — all boxes checked?

## 🚨 STUCK? TROUBLESHOOTING

| Symptom | Fix |
|---|---|
| 0 Calendly bookings after Week 2 | Your outreach isn't converting. Review DM scripts. Test new hooks. |
| Calls booked but no-shows | Send reminder 1 hr before. Add Calendly SMS reminders. |
| Calls go well but no proposals requested | You're not asking for the sale. Review Discovery Phase 5. |
| Proposals sent but no closes | Start `PROPOSAL_FOLLOWUP_SYSTEM.md`. Add `BEFORE_AFTER_DEMO.html` to calls. |
| Closed a deal but can't deliver | Follow `PROJECT_MANAGEMENT_SOP.md`. Start with Phase 1. |
