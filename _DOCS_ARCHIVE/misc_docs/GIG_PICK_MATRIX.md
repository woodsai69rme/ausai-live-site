## Scoring rule (use before you write the first DM)

For each candidate opportunity, score 0–3 on:

- **C** = "Can I deliver in <72 hours?" (build/existing assets)
- **D** = "Does the prospect have a Decision-maker identified?"
- **M** = "Is MY Motivation for this work above 6/10?"
- **$** = "Will the invoice clear > A$400 in one shot?"

Reject any opportunity with **C × D < 4**. Take any with **C × D × M ≥ 18**.

## PICK YOUR FIRST GIG — Daily rule

**Week 1** — push all 5 templates. Tag every reply with the path number.
**End of week 1**: which path got the *most replies* AND the *most "send the package"*? That's **August's path.**

If two paths tie, choose the one with the **higher per-gig A$** you listed above.

**End of month 1**: which path produced the **first accepted invoice**? That
becomes month-2's only path. Double down until A$5k cumulative.

## Anti-pattern (do not do)

- "I'll figure out which path once I get a client" → no, you start with a path.
- "I'll start a new path every week" → no, you finish first.
- "Build a website first" → Carrd for the gig, not WordPress.
- "Wait until the templates are perfect" → ship the first DM in <5 min.

## One-line decision rule

> If you can't ship the gig **today** from assets you already have, it's
