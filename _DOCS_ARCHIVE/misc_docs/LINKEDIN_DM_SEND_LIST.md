## Follow-Up Templates

### If they say "send" / "pack" / "sheet":
```
Thanks — link coming. While you're looking, two questions:
(1) what's broken right now that you'd fix today if you had extra hands?
(2) budget — credit-card expense or procurement cycle?
```

### If they say "$" / "how much" / "send proposal":
```
Got it — quick scope call (free, 15 min). I'm open Tuesday/Thursday 3–5pm AEST. Pick: [your Calendly link]
```

### If they say "not now" / "too busy":
```
No worries — I'll check in next quarter. In the meantime, here's the free checklist/cheatsheet: [link]. Feel free to reach out whenever it's relevant.
```

### If they don't respond after 5 days:
```
Hi {first_name} — just following up on my note about [topic]. Still happy to share the free [checklist/cheatsheet] if useful. No pressure either way.
```
