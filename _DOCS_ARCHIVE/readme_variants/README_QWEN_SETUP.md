## 🚨 Common Issues & Quick Fixes

| Problem | Solution |
|---------|----------|
| `qwen` command not found | Use full path: `X:\.npm-global\qwen.cmd` |
| Skills not loading | Restart terminal, check `dir ~/.agents/skills` |
| MCP server errors | Run manually: `npx -y @modelcontextprotocol/server-[name]` |
| Slow performance | Reduce `maxConcurrentTools` to 3 in settings.json |
| Skill not triggering | Use exact skill name in prompt |
| Authentication failures | Check environment variables |

**Full troubleshooting:** `QWEN_CODE_SETUP_DOCUMENTATION.md` → Troubleshooting Guide
