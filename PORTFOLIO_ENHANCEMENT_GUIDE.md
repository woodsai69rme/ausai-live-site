# 📸 PORTFOLIO ENHANCEMENT GUIDE — AusAI Tech

**Purpose:** Turn your portfolio from "here's what I did" to "here's why you should hire me."  
**Time:** 2-4 hours to upgrade all portfolio items.  
**Goal:** 5 polished portfolio pieces that close deals.

> **Why this matters:** A portfolio with before/after screenshots + measurable results gets 3× more inquiries than a list of projects.

## 1. THE BEFORE → AFTER FRAMEWORK

Every portfolio piece must answer 3 questions in 5 seconds:
- What was broken? (Before)
- What did you build? (The work)
- What changed? (After — with numbers)

### Template

```
Title: {Industry} Automation — {Specific Outcome}

THE PROBLEM:
{Client} was spending {X} hours/week on {manual process}.
Errors happened {Y} times/month. The team felt {emotion}.

THE SOLUTION:
I built a {tool_stack} automation that:
→ {Action 1}
→ {Action 2}
→ {Action 3}

THE RESULT:
✅ {Number} hours saved per week
✅ {Number}% fewer errors
✅ Team now focuses on {higher_value_work} instead
✅ Payback period: {X} weeks

[Before screenshot] ← side-by-side → [After screenshot/dashboard]
```

## 2. SCREENSHOT BEST PRACTICES

| Do | Don't |
|---|---|
| Use real outputs (anonymised) | Use stock photos or generic images |
| Show dashboards, workflow diagrams | Show code or terminal windows (unless for dev clients) |
| Highlight the key number (big, red arrow) | Clutter with too many annotations |
| Before = messy spreadsheet. After = clean dashboard. | Before = blank page. After = blank page. No contrast. |
| 1200×800px minimum | Blurry, low-res screenshots |

### Tools for Screenshots
- **Greenshot** (free, Windows) — annotate with arrows + highlights
- **CleanShot X** (paid, Mac) — same, plus scrolling capture
- **Canva** — overlay text + brand colours on screenshots

## 3. FORMATS FOR DIFFERENT PLATFORMS

| Platform | Best Format | Dimensions | File Type | Max Size | Notes |
|---|---|---|---|---|---|
| **Your website** | Before/after slider or side-by-side image | 1200×800px | PNG or WebP | <500KB | Use WebP for faster loading. Add alt text with target keywords. |
| **Upwork portfolio** | Single annotated screenshot + 3-bullet caption | 1000×750px | PNG or JPG | Upwork auto-compresses | Title max 50 chars. Caption max 500 chars. Tag with skills. |
| **Fiverr gallery** | 3-image carousel: problem → solution → result | 1280×769px | JPG | <5MB | Upload 3 images max per gig. First image = thumbnail (most important). |
| **LinkedIn featured** | PDF case study or carousel post | 1080×1080px (carousel) | PDF or PNG | PDF <100MB | Pin your best case study to "Featured" section. |
| **Discovery call deck** | Single slide: before/after + 3 numbers | 1920×1080px (16:9) | PNG | <2MB | Screen-share one slide. Don't show a 10-slide deck. |
| **Proposal attachment** | 2-pager PDF: problem, solution, result, testimonial | A4 (210×297mm) | PDF | <5MB | Attach to proposal email. "See attached example of similar work." |
| **Twitter/X** | Before/after side-by-side image | 1600×900px | PNG or JPG | <5MB | Add alt text. Pin to profile. |
| **Email signature** | Small thumbnail linking to full portfolio | 300×200px | PNG | <100KB | Link to your portfolio page or case study. |
| **Instagram** | Carousel (5 slides): problem-architecture-build-result-testimonial | 1080×1080px | PNG or JPG | — | Swipe-through format = higher engagement. |

### Platform Upload Checklist

- [ ] **Your website:** 5 portfolio items with before/after + results
- [ ] **Upwork:** 5 portfolio items tagged with skills
- [ ] **Fiverr:** 3 gallery images per gig (9 total for 3 gigs)
- [ ] **LinkedIn featured:** 1-2 best case study PDFs
- [ ] **Discovery deck:** 1 master slide (update per client industry)
- [ ] **Proposal template:** Link to portfolio or attach PDF

## 4. VIDEO PORTFOLIO PIECES

A 30-second screen recording beats 10 screenshots.

### The 30-Second Portfolio Video

```
0:00-0:05: "This client spent 15 hours/week doing THIS." (show manual process)
0:05-0:15: Screen recording: the automation running
0:15-0:25: "Result: 15 hours → 15 minutes. 94% fewer errors."
0:25-0:30: Your logo + CTA
```

**Tools:** OBS (free) for screen recording. CapCut (free) for editing+captions.

## 5. IF YOU DON'T HAVE REAL CLIENTS YET

Build demo portfolio pieces. Label them honestly.

```
"Demo: E-Commerce Lead Automation" (mock data, real architecture)

Don't:
- Fake client names
- Invent testimonials
- Use real company logos without permission

Do:
- Build real demos that work
- Label as "Demo — Architecture Example"
- Update to real once you have actual clients
```
